## How can I execute it?

1. Check if you have the [mysql](https://www.mysql.com/downloads/)(8 or newest), [JDK java](https://www.java.com/pt-BR/download/manual.jsp)(8 or newest)
2. Configure the application.properties
3. Execute your code
